# 💰 Sistema Financeiro em Python

Projeto de um sistema financeiro básico desenvolvido em Python com integração a banco de dados MySQL.

## 📚 Objetivo

Este projeto foi criado com o objetivo de:

- Praticar lógica de programação
- Aplicar conceitos de banco de dados relacional
- Integrar Python com MySQL
- Evoluir de scripts simples para sistemas mais estruturados

## 🚀 Funcionalidades

- Criar conta
- Consultar saldo
- Realizar depósitos
- Realizar saques
- Visualizar histórico de transações

## 🧠 Conceitos aplicados

- CRUD (Create, Read, Update, Delete)
- Banco de dados relacional
- Chaves primárias e estrangeiras
- Manipulação de dados com Python
- Estruturação de sistema com regras de negócio

## 🛠️ Tecnologias utilizadas

- Python
- MySQL

## 📊 Estrutura do Banco de Dados

### Tabela: contas

- id_contas (PK)
- nome_contas
- saldo_contas

### Tabela: transacoes

- id_transacao (PK)
- id_contas (FK)
- tipo_transacoes
- valor_transacao
- data_transacao
